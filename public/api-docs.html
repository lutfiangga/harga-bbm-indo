Selamat datang di dokumentasi API Harga BBM Indonesia. Anda dapat menggunakan endpoint ini untuk mengambil data harga bahan bakar terkini secara real-time dari Pertamina, Shell, BP, Vivo, dan Mobil Indostation.

GET /api/prices

Mengambil data harga BBM terbaru. Mendukung filtering berdasarkan provinsi dan provider.

Query Parameters

Parameter Tipe Deskripsi
provider string Filter provider (e.g. pertamina, shell)
provinceId string ID Provinsi (dari endpoint /api/provinces)

Response Example

{
  "success": true,
  "data": {
    "lastUpdated": "2026-01-05T09:00:00.000Z",
    "providers": {
      "pertamina": [
        {
          "province": "DKI Jakarta",
          "products": {
            "Pertalite": 10000,
            "Pertamax": 13500
          }
        }
      ]
    }
  }
}
GET /api/provinces

Mengambil daftar provinsi di Indonesia (sumber: Emsifa API).

Response Example

{
  "success": true,
  "data": [
    { "id": "11", "name": "ACEH" },
    { "id": "12", "name": "SUMATERA UTARA" }
  ]
}
POST /api/refresh

Memaksa server untuk melakukan scraping ulang data terbaru (Bypass Cache).

Response Example

{
  "success": true,
  "message": "Data refreshed successfully"
}
